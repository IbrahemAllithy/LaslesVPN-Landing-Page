# LaslesVPN Landing Page

This is a responsive landing page for **LaslesVPN**, built completely from scratch based on a [Figma Community Design](https://www.figma.com/design/y4g7B9BSJsuPkI101iyF5E/FREEBIES-Landingpage-LaslesVPN--Community-?node-id=1-127).

## 🚀 Features

- **Pixel-Perfect Design**: Accurately matches the provided Figma design.
- **Zero JavaScript**: Built entirely without JavaScript to demonstrate advanced CSS capabilities.
- **CSS-Only Interactions**:
  - Fully functional responsive mobile menu using the **Checkbox Hack**.
  - Interactive Testimonials Carousel using **CSS Scroll Snapping**.
  - Smooth hover effects and transitions across buttons, pricing cards, and links.
- **Fully Responsive**: Optimized for Desktop, Tablet, and Mobile devices using Flexbox and CSS Grid.
- **Semantic HTML5**: Well-structured and accessible HTML markup.
- **Organized CSS**: Clean and modular CSS utilizing custom properties (variables) for easy theme management.

## 🛠️ Technologies Used

- **HTML5** (Structured in `index.html`)
- **CSS3** (Separated in `css/style.css`)

```text
final-project-LaslesVPN/
```

## 💻 How to Run

1. Clone or download this repository.
2. Navigate to the project folder.
3. Open `index.html` in any modern web browser.
*(No local server, build tools, or dependencies are required!)*
